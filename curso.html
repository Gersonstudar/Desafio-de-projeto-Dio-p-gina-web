Falando sobre tags

Ao início abordado o conteúdo anterior trabalhado e a partir disso explica os elementos básicos como títulos etcs e que isso são tags pesquisando sobre de acordo com o site Hcode (2022) as tags são etiquetas, marcadores que indicam onde um elemento começa e onde termina ou seja ele determina a estrutura e as características da página que serão trabalhadas.

Alguns exemplos utilizados foram as tags

  • i para formatar o texto em itálico.
  • strong para formatar o texto em negrito.
  • input input na sua utilização aparece um campo para interações e junto dela você determinar qual atributo “conteúdo que mais a frente será melhor abordado” será utilizado se for texto então type text, ou números type number.
  • img para adicionar imagens a página.

Atributo das tags

Neste tópico é explicado sobre os atributos onde seriam um tipo de propriedade que a tag pode ter exemplos usados.

  • id atribuído para identificar no Java script qualquer elemento que será usado a partir de determinado nome.
  • style voltado para o CSS onde pode adicionar cor ou mudar a fonte etc.
  • class que faz com que uma classe e todos as tags que possuírem o nome desta classe serão alteradas, se refere a um conjunto com título em comum.
  • type atributo dedicado para a tag input ele determina o tipo de estrutura que será inserida na caixa do input seja texto, número ou até mesmo cor.

Links

Navegar na internet nada mais é do que mudar de página por página utilizando links, no HTML a tag utilizada para isso é a tag a fazendo referência a palavra âncora e dentro dela é adicionado o atributo href que significa “referência” ou seja o endereço a ser utilizado vale destacar que após href é necessário adicionar um título ao link.

Após adicionar um link o programador escolhe se a tag será aberta numa nova aba ou na mesma onde o site está, para isso utiliza-se o atributo target após o link do href você escolhe _self para continuar na mesma aba ou _blank para adicionar uma nova aba com o novo conteúdo, vale destacar mesmo que o programador não use a propriedade _self, de maneira automática, o arquivo irá continuar na mesma aba, só muda se utilizar _blank.

Por fim possível adicionar um título que aparece situar o mouse em cima do link isto acontece quando se é utilizado após o href o atributo title, ao utilizar isso uma pequena Caixa de informação aparecerá na tela do usuário mostrando o título que foi adicionado.
